Loops
for → fixed loop
while → condition loop
do...while → runs at least once
forEach → array loop
for...of → values
for...in → index
break → stop
continue → skip

for loop
 Used when you know how many times to run

for (let i = 1; i <= 5; i++) {
    console.log(i);
} Output: 1 2 3 4 5

 while loop
Runs until condition becomes false

let i = 1;
while (i <= 5) {
    console.log(i);
    i++;
}
 do...while loop
 Runs at least one time

let i = 1;
do {
    console.log(i);
    i++;
} while (i <= 5);

forEach
Used for arrays
let arr = [1, 2, 3];
arr.forEach(x => console.log(x));

break & continue
break;    // stop loop
continue; // skip one step

Template Literals
Template literals are used to create strings easily
backticks → ` ` (not " ")

Functions
A function is a block of code that you can reuse whenever needed
function add(a, b) {
    return a + b;
}

Arrow Function
let add = (a, b) => a + b;


Math.random() → generates random number between 0 and 1
Math.floor(Math.random()*100)+1 → random number from 1 to 100
arr.indexOf(x) → returns index of element, if not found returns -1
arr.includes(x) → checks element exists or not (true/false)
arr.push(x) → adds element at end
arr.pop() → removes element from end
arr.unshift(x) → adds element at beginning
arr.shift() → removes element from beginning
arr.splice(i,n) → removes/modifies elements from index

typecast

parseInt(x) → converts to integer (removes decimal)
parseFloat(x) → converts to decimal number
+x → shortcut to convert string to number
